Nuestro Producto Estrella

Características

  • Alta Sensibilidad: Detecta niveles de radiación UV desde 0 hasta 11+ en la escala UV, con precisión de ±0.5 UVI.
  • Resistencia al Agua: Certificación IP65 que permite su uso en diversos ambientes meteorológicos.
  • Pantalla LCD: Muestra lecturas claras con código de colores para facilitar la interpretación de los niveles de radiación.
  • Portátil y Ligero: Pesa solo 150g con baterías incluidas, ideal para uso en campo.

Descripciones

  • Uso Meteorológico: Perfecto para estaciones meteorológicas profesionales y aficionadas.
  • Aplicaciones Educativas: Ideal para demostraciones científicas y proyectos escolares.
  • Medición Precisa: Utiliza sensor fotoeléctrico de silicio de última generación para lecturas confiables.
  • Calibración Sencilla: Sistema de auto-calibración que mantiene la precisión del instrumento.
  • Compatibilidad: Puerto de datos para conexión a sistemas de registro continuo.

Funcionamiento

El Detector UV funciona mediante un sensor fotoeléctrico de silicio especializado que convierte la radiación ultravioleta en señales eléctricas medibles. Cuando la luz UV incide sobre el sensor, genera una corriente proporcional a la intensidad de la radiación presente. Esta señal es procesada por el microprocesador interno y convertida a la escala estándar del Índice UV (UVI).

Para utilizarlo, simplemente exponga el sensor (ubicado en la parte superior del dispositivo) directamente hacia el cielo o la fuente de radiación UV que desea medir. En pocos segundos, la pantalla LCD mostrará el valor medido junto con una indicación de color que clasifica el nivel de riesgo (verde para bajo, amarillo para moderado, naranja para alto y rojo para extremo).

El dispositivo incluye funciones adicionales como registro de máximos diarios, promedio de exposición y alertas configurables para niveles peligrosos. La memoria interna permite almacenar hasta 30 días de mediciones con intervalos programables.
